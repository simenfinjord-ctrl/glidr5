# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Glidr, please report it responsibly by emailing:

**simen@glidr.no**

Please include a description of the issue, steps to reproduce, and any relevant details.

## Disclosure Policy

- Do not publicly disclose the vulnerability until a fix has been deployed.
- We will acknowledge your report promptly and keep you informed of our progress.
- Once the fix is live, we are happy to credit responsible disclosers if desired.

Thank you for helping keep Glidr secure.
